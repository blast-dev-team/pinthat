# 협업 워크플로우

> 글로벌 규칙 `~/.claude/CLAUDE.md` (G-1~G-11)을 기반으로 합니다.

## 역할

| 역할 | 담당 | 주요 업무 |
|------|------|-----------|
| 대표 | 이연진 | 방향 결정, 최종 승인 |
| 코워커 (QA/PM) | Cowork AI | PRD, 스펙, QA, 검수 |
| 클로드 코드 (개발) | Claude Code | 코드 구현, 버그 수정 |

## 일반 개발 흐름

```
코워커: DEV-REQUEST.md에 스펙 등록
  ↓
클로드 코드: 구현 → hooks 실행 → 🔵 개발완료
  ↓
코워커: 검수 (PRD 대조 + 보안 체크 + 본질 체크) → 🟢 검수완료
```

## QA 흐름

```
코워커: QA.md에 이슈 등록 (🔴)
  ↓
클로드 코드: 수정 → hooks 실행 → 🔵 수정완료
  ↓
코워커: 검증 (코드 확인 + 브라우저 테스트) → 🟢 검증완료
```
