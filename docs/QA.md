## 이슈 목록

### QA-001: 자유 위치 이동 — 메모 입력란 미작동
- **상태**: 🔵 수정완료
- **발견일**: 2026-04-02
- **화면**: 자유 위치 이동 피드백 모달
- **재현 방법**: QA 모드 활성화 → 요소 클릭 → "자유 위치 이동" 타입 선택 → 메모 입력란 클릭 후 타이핑 시도
- **기대 동작**: 메모 입력란에 텍스트 입력 가능 (예: "사이드바 아래로 옮겨주세요")
- **실제 동작**: 입력란이 깨져서 텍스트 입력 불가 (클릭/포커스/입력 중 하나 이상 작동 안 함)
- **첨부**: 스크린샷 — 모달에 "출발/목적지 셀렉터" + 빈 textarea 표시됨
- **수정 내역**: "위치이동" 피드백 타입 전체 구현 — 📍 위치이동 탭 추가, 컴포넌트 이동(방향 선택) + 자유 위치 이동(crosshair 좌표 클릭) 서브메뉴, textarea 메모 입력 정상 동작, 마크다운 출력에 이동 정보 포함. z-index 99995~99996 crosshair 오버레이, pointer-events 분리, setTimeout focus 100ms 적용.
- **검증 결과**: (코워커 작성)
